# 一、当前任务文本的核心问题

---

# 问题 1：没有强制“全量遍历”

你现在：

```md

```

AI 会理解成：

```text

```

这是最致命问题。

你需要明确：

```md
必须遍历 sponsor_raw.json 中的全部记录。
禁止抽样。
禁止只扫描前 N 条。
```

这是工业级任务描述的核心。

---

# 问题 2：没有“执行统计”

AI 不知道：

* 总共有多少 sponsor
* 有多少含链接
* 有多少有效
* 有多少失效
* 有多少文章
* 有多少平台

于是它不会真正做数据分析。

必须要求输出：

```json
{
  "totalSponsors": 1328,
  "withLinks": 186,
  "validArticles": 64,
  "filteredOut": 122,
  "selected": 28
}
```

这是关键。

# 问题 3：没有强制“URL 验证”

现在：

```md

```

太模糊。

AI 很可能：

* 不打开
* 不检查状态码
* 不检查反爬
* 不检查 404

应该强制：

```md
必须实际访问每个 URL。
必须验证 HTTP 状态码。
必须确认正文包含 FlyEnv。
```

